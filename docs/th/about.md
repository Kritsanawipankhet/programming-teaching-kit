# About

**Programming Teaching Kit** is a collection of structured learning materials and resources for teaching and learning programming.

The content is designed to be clear, practical, and accessible. It can be used by independent learners, students studying with an instructor, or teachers looking for materials to support their lessons.

The website includes lessons, runnable code examples, exercises, skill checks, and small projects to help learners understand programming concepts and apply them through hands-on practice.

## Purpose

This project was created to:

* Organize programming lessons and teaching materials in one place
* Explain programming concepts in a clear and approachable way
* Provide code examples that learners can run and experiment with
* Offer exercises that develop problem-solving and programming skills
* Provide skill checks for assessing prior knowledge
* Support additional courses and technical topics in the future

## Who Is This For?

The content is suitable for:

* People who are new to programming
* Students who want to review programming fundamentals
* Learners who want additional coding practice
* Instructors looking for examples and exercises
* Anyone who prefers learning through hands-on practice

## What You Will Find

Each course may include:

* **Lessons** — Explanations of concepts presented in a structured order
* **Code Examples** — Runnable programs that demonstrate key concepts
* **Exercises** — Practice problems for developing programming skills
* **Skill Checks** — Short assessments for evaluating prior knowledge
* **Projects** — Practical tasks that combine concepts from multiple lessons

The structure of each course may vary depending on its topic, programming language, learning objectives, and intended difficulty level.

## Learning Approach

The materials follow a gradual, practice-oriented approach. Learners begin with fundamental concepts, explore examples, and then apply what they have learned through exercises and projects.

Learners are encouraged to write, modify, compile, and run the code themselves instead of only reading or copying examples. Encountering errors, investigating their causes, and correcting them are important parts of learning how to program.

## About the Author

This project is created and maintained by **Kritsana Wipankhet**, a software developer and programming instructor.

The materials are developed from practical experience in software development, software architecture, problem-solving, and teaching learners with different levels of technical knowledge.

## Source Code

The website content and public code examples are available on GitHub, allowing learners and instructors to inspect, run, and adapt the materials.
